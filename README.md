# Food Sharing App

A community-driven food sharing platform that connects people who want to share surplus food with those who need it, while ensuring user safety, consent, and privacy.

## ğŸ”„ Application Workflow

### 1. User Onboarding
- User registers an account
- User logs in
- Verification process (identity / trust verification)

### 2. Sharing Food
- User posts available food
- Location is shared only with user consent
- Interested users can request pickup
- Food is picked up after confirmation

### 3. Requesting Help
- User creates a help request
- Help details are visible, but sensitive data is hidden
- Only verified users can accept requests
- Once accepted:
  - Name, location, and important details are revealed
- Multiple users can help
- Successful help increases user karma
